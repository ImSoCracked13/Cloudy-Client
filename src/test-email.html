Email Verification Test

Note: This page helps you test and debug the email verification flow.

1. Test EmailJS Configuration

Run this code in your browser console to test if EmailJS is properly configured:

async function testEmailJS() {
  const serviceID = 'service_ege9r9v';
  const templateID = 'template_srq2imp';
  const userID = 'Ajpl74OWTk5bSN3zp';
  
  // The URL that should be used in the verification link
  const verificationURL = 'http://YOUR-ACTUAL-DOMAIN:3001/verify-email?token=test-token';
  
  try {
    console.log('Testing EmailJS...');
    
    // Parameters that match your template
    const params = {
      name: 'Test User',
      email: 'your-email@example.com',
      verification_url: verificationURL
    };
    
    console.log('Sending with params:', params);
    
    // Send test email - make sure emailjs is imported
    const result = await emailjs.send(serviceID, templateID, params, userID);
    console.log('Success!', result);
    return result;
  } catch (error) {
    console.error('Failed!', error);
    return error;
  }
}
